URL Shortener API

by QuanchiFootball


This is a simple API that will intake any url and generate a four digit shortcut code.


Example

Generating a new short code :

https://qcfb-urlshortener.glitch.me/new/https://www.facebook.com

Returns :

{
"originalurl":"https://www.facebook.com",
"shortcut":"https://qcfb-urlshortener.glitch.me/1234"
}

New URL inputs must include the appropriate protocol (http://, https://, or ftp://) or else the output object will return an error message.


Shortcut code usage :

https://qcfb-urlshortener.glitch.me/1234
